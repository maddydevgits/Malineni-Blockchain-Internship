# Book Hotel Rooms 
Book Hotel Rooms using Blockchain is a project which has a dapp built on Blockchain with Truffle suite.

# Softwares
1. Ganache
2. NodeJs
3. Truffle Suite
4. Python3
5. VS Code

# Packages
1. pip install web3
2. pip install flask

# Process
1. Install Softwares
2. Install Packages
3. Clone Repo
4. Migrate Smart Contracts
5. cd src, python app.py
